Instalación automatizada

Selecciona tu entorno y copia el script verificado para desplegar la herramienta.

BASH
curl -sSL https://raw.githubusercontent.com/farllirs/xzp-slash/master/install.sh | bash
BASH
echo "deb [trusted=yes] https://farllirs.github.io/xzp-slash/ ./" | sudo tee /etc/apt/sources.list.d/xzp-slash.list sudo apt update && sudo apt install xzp-slash
BASH
echo "deb [trusted=yes] https://farllirs.github.io/xzp-slash/ ./" > $PREFIX/etc/apt/sources.list.d/xzp-slash.list pkg update && pkg install xzp-slash

Potente por diseño

Optimiza tus flujos de trabajo en servidores o entornos locales sin configuraciones densas.

Multiplataforma

Soporte nativo para las distribuciones de Linux más empleadas y arquitecturas ARM a través de Termux.

Detección Contextual

Analiza de forma reactiva la presencia de repositorios Git o entornos de Node.js y Python en el directorio actual.

Fluidez Interactiva

Administrador de tareas integrado y analíticas en tiempo real empaquetadas sobre la agilidad de fzf.

Comandos principales

slash
Lanzador principal interactivo. Escanea el contexto directo de carpetas para desplegar tareas rápidas.
slash -c
Abre un completo menú visual de configuración enfocado en el usuario. Desde aquí puedes modificar ajustes avanzados del sistema, tales como activar la inicialización o automatización automática del widget al presionar la tecla / (slash).
slash-task
Monitor interactivo de procesos en tiempo real. Gestiona hilos, consumo de CPU y memoria de forma visual.
slash-health
Diagnóstico rápido del sistema. Imprime un reporte limpio sobre almacenamiento, red y temperaturas críticas.
